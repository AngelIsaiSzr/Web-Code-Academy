¿De qué trata el curso?

Este curso te enseñará a programar en Java desde cero. Aprenderás los fundamentos del lenguaje, incluyendo su sintaxis básica y estructuras de control. También aprenderás sobre Programación Orientada a Objetos (POO) y cómo aplicarla para resolver problemas de programación. Además, aprenderás a desarrollar aplicaciones móviles Android y aplicaciones web utilizando Java.

Aprende todo sobre Java

Estos son los temas más importantes que aprenderás en el curso

Programación Orientada a Objetos

En este tema aprenderás los conceptos fundamentales de la programación orientada a objetos, como clases, objetos, herencia, polimorfismo, encapsulamiento, entre otros. Aprenderás cómo aplicarlos en Java y cómo utilizarlos para resolver problemas de programación.

Desarrollo de aplicaciones Android con Java

Java es uno de los lenguajes de programación principales para el desarrollo de aplicaciones móviles Android. En este tema aprenderás a utilizar Java para desarrollar aplicaciones Android, desde la creación de la interfaz de usuario hasta la integración con servicios web y bases de datos.

Desarrollo de aplicaciones Web con Java

En este tema aprenderás a desarrollar aplicaciones web utilizando tecnologías como Servlets, JSP, JSTL, JDBC, etc. Aprenderás a trabajar con bases de datos, crear formularios, manejar sesiones y cookies, y crear páginas dinámicas utilizando Java. También aprenderás sobre la arquitectura de las aplicaciones web.

Fundamentos de Java

En este tema aprenderás la historia y los fundamentos de Java, aprenderás tanto su sintaxis básica como quien lo creo, además de tipos de datos, operadores, estructuras de control, entrada y salida de datos, entre otros. Este tema es esencial y fundamental para cualquier programador que quiera aprender Java desde cero.

¿Cuáles son los requisitos para acceder al curso?

Este curso de Java está diseñado para ser accesible para cualquier persona interesada en aprender a programar en este lenguaje de programación. No se requiere ningún conocimiento previo para acceder al curso, sólo es necesario tener una actitud de aprendizaje y estar dispuesto a dedicar tiempo para practicar. Así que, si tienes la actitud correcta y estás listo para aprender.
¡Este curso es para ti!

  • Persistencia de datos con JDBC y JPA
  • Autenticación y seguridad en aplicaciones web
  • Despliegue de aplicaciones web en servidores de aplicaciones

Módulo 6: Programación avanzada en Java

En este módulo, se explorarán aspectos avanzados de la programación en Java. Los estudiantes aprenderán sobre programación concurrente y multihilo, lo que les permitirá desarrollar aplicaciones que realicen tareas simultáneas. También se cubrirá el manejo de colecciones y estructuras de datos avanzadas, así como el uso de expresiones lambda y programación funcional para escribir un código más conciso y legible. Los estudiantes también aprenderán sobre anotaciones y reflexión, lo que les permitirá aprovechar la flexibilidad y capacidad de extensión de Java. Por último, se introducirá el desarrollo de aplicaciones que utilizan API externas y la creación de plugins.


  • Programación concurrente y multihilo
  • Manejo de colecciones y estructuras de datos avanzadas
  • Expresiones lambda y programación funcional
  • Anotaciones y reflexión
  • Uso de API externas y desarrollo de plugins

Módulo 7: Desarrollo de aplicaciones empresariales con Java

Este módulo se centra en el desarrollo de aplicaciones empresariales utilizando Java. Los estudiantes aprenderán sobre las arquitecturas empresariales y cómo abordar los desafíos comunes en este tipo de aplicaciones. Se explorará el framework Spring, ampliamente utilizado en el desarrollo de aplicaciones empresariales, y se enseñará a utilizarlo para la creación de aplicaciones robustas y escalables. También se abordará el acceso a bases de datos mediante ORM (Object-Relational Mapping), lo que facilita la persistencia de datos en aplicaciones empresariales. Los estudiantes también aprenderán sobre seguridad y autenticación en aplicaciones empresariales, y se introducirán técnicas de pruebas unitarias y despliegue de aplicaciones para garantizar la calidad y disponibilidad de los sistemas desarrollados.


  • Introducción a las arquitecturas empresariales
  • Desarrollo de aplicaciones con el framework Spring
  • Acceso a bases de datos mediante ORM (Object-Relational Mapping)
  • Seguridad y autenticación en aplicaciones empresariales
  • Pruebas unitarias y despliegue de aplicaciones

Módulo 8: Proyecto Final - Desarrollo de una aplicación Java

En este módulo, los estudiantes tendrán la oportunidad de aplicar todos los conocimientos adquiridos a lo largo del curso para desarrollar un proyecto final. Trabajarán en equipo para crear una aplicación Java completa, desde la planificación y diseño hasta la implementación y prueba. El proyecto final les permitirá consolidar sus habilidades de programación en Java y demostrar su capacidad para desarrollar aplicaciones funcionales y de calidad.


1. Planificación del proyecto

  • Definición de los requisitos y funcionalidades del proyecto
  • Creación de un plan de desarrollo y asignación de tareas

2. Diseño de la aplicación

  • Creación de diagramas de clases y diagramas de secuencia
  • Diseño de la interfaz de usuario

3. Implementación del proyecto

  • Desarrollo de las funcionalidades principales de la aplicación
  • Implementación de la lógica de negocio

4. Pruebas y depuración

  • Realización de pruebas unitarias y de integración
  • Identificación y solución de errores y fallos

5. Mejoras y optimizaciones

  • Identificación de áreas de mejora en el código y rendimiento
  • Implementación de mejoras y optimizaciones

6. Documentación y presentación

  • Elaboración de documentación técnica del proyecto
  • Preparación de una presentación del proyecto final
