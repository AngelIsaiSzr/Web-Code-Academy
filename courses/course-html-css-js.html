Bienvenido al curso de Desarrollo Web FullStack

¿De qué trata el curso?

En este curso aprenderás los conceptos básicos y avanzados de HTML, CSS y JavaScript para crear sitios web interactivos y atractivos. Adquirirás conocimientos en el front-end para diseñar y desarrollar interfaces de usuario con animaciones, degradados y otros elementos visuales que hagan que tu sitio web se destaque.

También aprenderás sobre el back-end para implementar funcionalidades avanzadas y dinámicas en tu sitio web, como la gestión de bases de datos y la programación de aplicaciones web.

Aprende todo sobre el desarrollo web

Estos son los temas más importantes que aprenderás en el curso

Desarrollo Web Responsive

Aprende a crear sitios web que se adapten a diferentes tamaños de pantalla, garantizando una experiencia de usuario óptima en cualquier dispositivo. En este módulo aprenderás sobre las mejores prácticas de diseño y desarrollo para crear sitios web que se vean bien y funcionen correctamente en cualquier dispositivo, desde computadoras hasta teléfonos móviles.

JavaScript

Aprende a programar en JavaScript y descubre cómo puedes utilizarlo para crear interacciones dinámicas en tus sitios web. En este módulo aprenderás sobre la historia de JavaScript, quién lo fundó y por qué es tan importante para el desarrollo web. También aprenderás los conceptos fundamentales de JavaScript, como variables, funciones, objetos y eventos.

CSS3

Descubre cómo utilizar CSS3 para diseñar sitios web visualmente atractivos y con estilo. En este módulo aprenderás sobre la historia de CSS3, quién lo fundó y por qué es tan importante para el desarrollo web. También aprenderás los conceptos fundamentales de CSS3, como selectores, propiedades y valores, y cómo aplicarlos para crear diseños responsivos y modernos.

HTML5

Aprende todo sobre HTML5, el lenguaje de marcado utilizado para crear sitios web modernos y dinámicos. En este módulo aprenderás sobre la historia de HTML5, quién lo fundó y por qué es tan importante para el desarrollo web. También aprenderás sobre los elementos y atributos de HTML5 y cómo utilizarlos para estructurar y crear contenido en tus sitios web.

¿Cuáles son los requisitos para acceder al curso?

Para acceder a este curso y sacarle el máximo provecho sólo necesitas tener deseos de aprender y estar dispuesto a dedicar tiempo para practicar. No se requiere ningún conocimiento previo para acceder a este curso de desarrollo web. El curso está diseñado para ser accesible para cualquier persona interesada en aprender a crear sitios web desde cero. Los temas se explican de forma progresiva y sencilla, empezando desde los fundamentos.

Temario del curso

En este módulo, los estudiantes obtendrán una introducción fundamental a los conceptos básicos de la programación y los lenguajes de programación. Este módulo está diseñado para sentar las bases antes de adentrarse en los módulos principales del curso. Los estudiantes adquirirán una comprensión clara de los conceptos clave y los términos fundamentales necesarios para desarrollar habilidades en programación.


  • Introducción a la programación
  • Introducción a los lenguajes de programación
  • Glosario básico

Este módulo les permitirá avanzar con confianza en los módulos subsiguientes del curso y maximizar su aprendizaje.

Módulo 1: Introducción a HTML

En este módulo, los estudiantes aprenderán los fundamentos del lenguaje de marcado HTML (HyperText Markup Language). Explorarán la estructura básica de una página web, etiquetas HTML, atributos y elementos comunes. También comprenderán cómo crear una jerarquía de contenido y enlaces entre diferentes páginas. Pondremos los cimientos de tu carrera como desarrollador web al aprender a dominar a profundidad el lenguaje HTML, el bloque de construcción fundamental de todas las páginas web.


  • Descubrirás la historia de HTML5, quién lo fundó y por qué es tan importante para el desarrollo web
  • Aprenderás a diseñar la estructura básica y global de una página HTML con etiquetas head, body y más!
  • Verás cómo usar etiquetas semánticas como h1-h6, section, nav, aside, article, footer para darle significado y jerarquía a tu contenido además de sus beneficios.
  • Aprenderás a inserir y estilar texto usando p, strong, em, br.
  • Insertarás imágenes, videos, audio a tus páginas con etiquetas como img, iframe, embed, source, figure, figcaption.
  • Crearás listas ordenadas y desordenadas para organizar contenido de forma lógica usando ol, ul y li.
  • Diseñarás tablas complejas usando th, tr, td, colspan y rowspan.
  • Generarás formularios sofisticados usando input, select, textarea, fieldset, label, button para enviar datos al servidor.

También se explicará la importancia de la semántica en la estructura HTML para mejorar la accesibilidad y optimización para buscadores.

Módulo 2: Estilos CSS

En este módulo, los estudiantes se sumergirán en el mundo de CSS (Cascading Style Sheets) y aprenderán a dar estilo a las páginas web. Aprenderán a aplicar colores, fondos, fuentes, márgenes y bordes a elementos HTML. También explorarán técnicas de diseño avanzadas, como la creación de diseños en columnas y la utilización de selectores y pseudoclases. Los estudiantes aprenderán a crear diseños limpios y atractivos para sus sitios web.


  • Conocerás la historia de CSS, quién lo fundó y por qué es tan importante para el desarrollo web
  • Aplicaremos estilos a HTML usando una variedad de selectores.
  • Dominaremos cajas de modelo CSS para posicionamiento y diseño.
  • Utilizaremos floats, flexbox y CSS Grid para maquetar efectivamente.
  • Personalizaremos tipografías, colores, fondos y más.
  • Implementaremos animaciones y transiciones en CSS.

Al final de este módulo, podrás diseñar sitios web con estilos CSS profesionales.

Módulo 3: JavaScript desde 0

En este módulo, los estudiantes obtendrán una introducción práctica a JavaScript, un lenguaje de programación fundamental para el desarrollo web. Aprenderán los conceptos básicos, como variables, tipos de datos, operadores y estructuras de control. También se familiarizarán con el manejo de eventos y la manipulación del DOM (Document Object Model). También se enseñará a crear interactividad en páginas web utilizando JavaScript. Los estudiantes aplicarán estos conceptos en proyectos prácticos.


  • Declararemos variables, utilizaremos tipos de datos primitivos y estructurados.
  • Aprenderemos a utilizar funciones, flechas y ámbito léxico.
  • Utilizaremos condicionales if/else y bucles for/while para controlar el flujo.
  • Manipularemos arrays y objetos para almacenar y recuperar datos.
  • Registrar eventos del DOM en HTML para interactuar con el usuario.
  • Usaremos selectores DOM para acceder y modificar el DOM.

Al final del módulo, tendrás los fundamentos para crear funcionalidad JavaScript compleja y dinámica para tus sitios web.

Iniciar Módulo 3

Módulo 4: Fundamentos del Frontend

Este módulo se enfoca en brindar una visión general de los fundamentos del desarrollo frontend. Los estudiantes explorarán las herramientas y tecnologías modernas utilizadas en el desarrollo web, como preprocesadores CSS, frameworks de JavaScript y sistemas de gestión de paquetes. También se abordarán temas como el flujo de trabajo de desarrollo, control de versiones y optimización de rendimiento.

Módulo 5: Diseño responsivo

En este módulo, los estudiantes aprenderán a crear sitios web responsivos que se adapten a diferentes dispositivos y tamaños de pantalla. Explorarán técnicas y conceptos como media queries, rejillas CSS y flexbox para crear diseños flexibles y fluidos. Se aplicarán buenas prácticas de diseño para mejorar la experiencia del usuario y la visualización en diferentes dispositivos.

Módulo 6: Maquetación web semántica

En este módulo, los estudiantes aprenderán la importancia de la maquetación web semántica. Comprenderán cómo estructurar y organizar el contenido de manera significativa utilizando las etiquetas HTML adecuadas. Se explorarán los beneficios de la maquetación semántica, como la accesibilidad, la indexación en motores de búsqueda y la mantenibilidad del código.

Módulo 7: Fundamentos del Backend

Este módulo proporcionará una introducción a los fundamentos del desarrollo backend. Los estudiantes aprenderán los conceptos básicos relacionados con la arquitectura cliente-servidor, solicitudes y respuestas HTTP, API (Application Programming Interface) y cómo interactuar con bases de datos desde el backend. También se expondrán diferentes tecnologías y lenguajes comunes utilizados en el desarrollo backend.

Módulo 8: Programación orientada a objetos

En este módulo, los estudiantes serán introducidos a la programación orientada a objetos (POO). Aprenderán los principios fundamentales de la POO, como clases, objetos, propiedades y métodos. Explorarán cómo aplicar la POO en el contexto del desarrollo web y cómo beneficiarse de las ventajas que ofrece este enfoque en la organización y modularidad del código.

Módulo 9: Bases de datos para aplicaciones web

En este módulo, los estudiantes adquirirán conocimientos básicos sobre bases de datos y su aplicación en el desarrollo web. Se explorarán conceptos como la creación y gestión de bases de datos, consultas básicas utilizando SQL (Structured Query Language) y cómo interactuar con bases de datos desde el backend de una aplicación web.

Módulo 10: Buenas prácticas de desarrollo web

En este módulo, se pondrá énfasis en las mejores prácticas en el desarrollo web. Los estudiantes aprenderán sobre la optimización del rendimiento, la seguridad, la accesibilidad y la mantenibilidad del código. También se discutirán conceptos como pruebas y depuración, control de versiones, documentación y trabajo en equipo.

Módulo 11: Proyecto final - Creando tu portafolio

En este módulo, los estudiantes aplicarán todos los conocimientos adquiridos a lo largo del curso en un proyecto final. Crearán su propio portafolio web, que les permitirá mostrar sus habilidades y proyectos a posibles empleadores o clientes. Este módulo les brindará la oportunidad de poner en práctica todo lo aprendido y exhibir su trabajo como desarrolladores web.


Se utilizará HTML semántico, CSS Grid o Flexbox, diseño responsivo y buenas prácticas para crear:


  • Una página de inicio bien estructurada y accesible
  • Una sección sobre mí con biografía y footer
  • Una sección de habilidades con lista de tecnologías
  • Una sección de proyectos con enlaces a muestras del trabajo
  • Un formulario de contacto funcional

Los estudiantes implementarán pruebas automáticas, uso de git, y documentarán su código.

La habilidad de crear un portafolio les ayudará a mostrar su trabajo y conseguir empleo como desarrolladores junior.
