Bienvenido al curso de Python FullStack

¿De qué trata el curso?

Este curso te enseñará a programar en Python desde cero. Comenzaremos explicando los fundamentos del lenguaje para luego aprender técnicas avanzadas con herramientas como módulos, POO y testing. Finalmente, aplicarás tus conocimientos creando proyectos prácticos como juegos, scripts útiles y aplicaciones gráficas con herramientas como Pygame, Matplotlib y Tkinter.

Aprende todo sobre Python

Estos son los temas más importantes que aprenderás en el curso

Proyectos prácticos

Aplicarás todos los conceptos de Python que hayas aprendido a lo largo del curso creando una variedad de proyectos prácticos independientes. Estos proyectos abarcarán desde aplicaciones sencillas como scripts y juegos, hasta aplicaciones web interactivas que utilicen herramientas como Tkinter, Pygame, Matplotlib, API's y más.

Programación Orientada a Objetos

Aprenderás los conceptos fundamentales de POO en Python como clases, atributos, métodos, herencia, polimorfismo e inicialización de objetos. Comprenderás cómo utilizar estos pilares de POO para organizar y reutilizar tu código, haciéndolo más legible, mantenible y escalable.

Métodos y Funciones avanzados

Estudiarás Métodos como map(), filter() y reduce(), funciones lambda, comprehensión de listas y tuplas, argumentos variables ( *args y **kwargs), además de funciones anónimas internas y decoradores para extender el alcance y la reutilización de funciones.

Fundamentos de Python

Aprenderás sobre la sintaxis única de Python, incluyendo tipos de datos incorporados, funciones integradas, manejo de errores, bucles, condicionales, operadores lógicos, ámbito de variables y más. Estos conceptos básicos formarán la base de tu aprendizaje en Python.

¿Cuáles son los requisitos para acceder al curso?

Este curso está destinado para cualquier persona interesada en aprender a programar. No se necesita ningún conocimiento previo para acceder al curso, sólo es indispensable tener una actitud de aprendizaje y estar dispuesto a dedicar tiempo para practicar. Así que, si tienes interés en aprender a programar en Python y estás listo para dedicar tiempo y esfuerzo a tu aprendizaje. ¡Este curso es ideal para ti!

Módulo 5: Módulos y paquetes

Aprenderás a organizar mejor tu código de Python al crear tus propios módulos y paquetes, luego importarlos desde otros scripts. Esto te permitirá evitar duplicar código y mantener tu programa principal limpio estructurando el código en una función lógica. También descubrirás PIP, el sencillo pero poderoso administrador de paquetes de Python que te permitirá instalar bibliotecas externas con sólo unos pocos comandos, ampliando enormemente las posibilidades de Python.


  • Importación de módulos
  • Creación de módulos y paquetes
  • Instalación de paquetes con PIP

Módulo 6: Excepciones y depuración

Dominarás dos de las herramientas más valiosas para resolver errores y crear código robusto en Python: manejo de excepciones y depuración. Explorarás decenas de increíbles librerías de código abierto construidas por la comunidad de Python, tanto incorporadas en su biblioteca estándar como disponibles externamente.
